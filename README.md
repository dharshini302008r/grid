
# 🔹 What is Grid? 📊
. Bootstrap grid divides the page into 12 columns
. Helps to arrange content in a clean and structured way 🧱
. Works for all devices (mobile 📱, tablet 💻, desktop 🖥️)
🔹 Structure of Grid 🏗️

👉 Basic order:
. Container 📦 → Row ➖ → Column 📊 → Content 🧾

✅ Explanation:
. Container 📦 → Main wrapper (centers content & adds spacing)
. Row ➖ → Creates horizontal section
. Column 📊 → Divides row into parts (out of 12)
